Dúvidas

Perguntas frequentes

Precisa saber programar para usar? +
Não. O Fissão é um app web simples — você acessa pelo navegador do celular ou computador e começa a usar imediatamente. Nenhuma instalação necessária.
Meus dados ficam salvos se eu trocar de celular? +
Sim. Tudo é salvo na nuvem (Supabase) e sincronizado automaticamente entre dispositivos. Faça login em qualquer lugar e seus dados aparecem na hora.
O Atominho realmente usa meus dados ou é genérico? +
Ele usa seus dados reais — taxa de acerto, pontos fracos, revisões pendentes, streak e muito mais. Quanto mais sessões você registrar, mais preciso ele fica.
Funciona para qualquer vestibular ou só ENEM? +
O foco principal é o ENEM — o mapa de incidência e os temas são calibrados para a prova. Mas o sistema de sessões, revisões e Anki funciona para qualquer vestibular.
Como cancelo? +
Basta entrar em contato pelo WhatsApp. Não há fidelidade nem multa. Cancelando, você mantém acesso até o fim do período pago.
